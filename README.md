# 字节跳动青训营—抖音项目

## 项目运行
1. 将TikTok-Backend/model/common/videoCommon.go/中的VIDEO_RUL_MID更改为自己的IP地址
2. 在终端运行main.go文件
3. 将抖声app中的BaseUrl更改为http://x.x.x.x:8080，x.x.x.x为你的IP地址
4. 重启并保存后，即可运行。
