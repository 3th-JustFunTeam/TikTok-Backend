# TikTok-Backend
TikTok backend in Golang.
## 项目结构
API文档见 docks/API.md
## 部署
1. 在Linux服务器安装Docker, Docker-compose
```shell
$ sudo apt update
$ sudo apt install docker 
$ sudo pip install docker-compose
```
2. 运行项目
```shell
$ docker-compose up -d
```
